# BobCAD-CAM Custom Post Scripts

Custom canned cycle definitions and helper logic for use in BobCAD-CAM post processors.

## What's included
- Custom canned cycle definitions (e.g. drilling, tapping, boring — name the actual ones)
- tool change handling, custom output formatting

## Machine types
Mill / Lathe / Mill-Turn

## Requirements
- BobCAD-CAM version 31 or later

## Usage
Copy the relevant script/function into your post processor file
